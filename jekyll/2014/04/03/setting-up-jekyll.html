setting up jekyll

03 Apr 2014

Jekyll was super easy to get started with, following their quick start.

$ [sudo] gem install jekyll
$ jekyll new blog

and that’s it! After playing with it for a bit, a had a few problems:

  • vim didn’t like the filter syntax in the markdown files
  • I imagine I’ll want to use LaTeX at some point, but that’s not set up by default.

Luckily, I’m not the first to have these problems. After some googling, I found Tim Pope’s vim-liquid, for proper syntax highlighting of liquid markdown, and Cecil Woebker’s blog post to set up MathJax. 1 Now I can include content like \(a^2 + b^2 = c^2\).

I expect I’ll post a lot of gists, so I installed mattn’s gist-vim as well.

  1. Now I’m using pandoc, via David Sanson’s pandoc jekyll plugin↩
